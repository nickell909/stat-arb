# Crypto Arbitrage Monitor

Приложение для мониторинга цен и анализа исторических данных криптовалютных бирж с поддержкой статистического арбитража (парный трейдинг).

## Структура проекта

```
├── src/
│   ├── connectors/     # WebSocket и REST коннекторы к биржам
│   ├── api/            # REST API для загрузки исторических данных
│   ├── data/           # Работа с базой данных и нормализация
│   ├── services/       # Бизнес-логика и сервисы
│   ├── models/         # Модели данных
│   └── utils/          # Утилиты и вспомогательные функции
├── tests/              # Тесты
├── config/             # Конфигурационные файлы
└── requirements.txt    # Зависимости
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python src/main.py
```

## Модуль 1 — Сбор данных

### 1.1 WebSocket-коннекторы (real-time)
- Поддержка Binance, Bybit, OKX
- Автоматический реконнект с exponential backoff
- Heartbeat-пинг каждые 30 секунд
- Логирование задержек (latency)
- Очередь сообщений с буфером

### 1.2 REST API — исторические данные
- Загрузка OHLCV-свечей
- Умная загрузка (только недостающие данные)
- Параллельная загрузка с соблюдением rate limits

### 1.3 Нормализация данных
- Единая схема данных
- Обработка выбросов
- Заполнение пропусков (forward-fill)
